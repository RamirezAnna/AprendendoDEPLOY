# Página de Cadastro — AprendendoDEPLOY

Esta pasta contém uma página HTML simples de cadastro criada como exemplo.

Arquivos:
- `signup.html` — formulário de cadastro com campos Nome, Email e Senha. Estilo inspirado na imagem fornecida.

Como usar:
- Abra o `signup.html` no navegador (duplo clique) ou use Live Server no VS Code.
- O formulário tem validação básica e botão para mostrar/ocultar senha.
